# Skill: Integrate With Existing Code

## When to Use

Use this skill when adding a feature into an existing codebase.

## Steps

1. Convert the user request into a micro-spec.
2. Generate or read the code-map.
3. Filter the code-map by entity, operation, module, route, and call depth.
4. Inspect the selected source files.
5. Identify the established local pattern.
6. Apply changes using the relevant coding skill.
7. Run validation.
8. Update code-map and project-patterns if the change introduces an intentional new pattern.

## Key Rule

Do not rely on code-map alone for implementation. Always inspect source code before editing.
