# Worked Example: Leave Management

[← Home](../Home.md) | [Next: Starting Context](01-starting-context.md)

This worked example demonstrates how structured AI-assisted engineering can be applied to a small existing codebase scenario.

The example is intentionally small. The goal is not to build a full application, but to show how the models, use cases, and AI context fit together.

## Scenario

We have an existing Leave Management application.

The application already supports:

- employees submitting leave requests
- managers viewing pending requests

A new change request arrives:

```text
Allow managers to approve or reject leave requests.
```

## What This Example Demonstrates

The example walks through:

1. Starting context
2. Required models
3. AI context initialization
4. Change request handling
5. Validation and completion report
