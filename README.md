# AI for Software Engineering (AI-for-SE)

This repository explores how software engineers use AI in practice, the limitations they encounter, and how more structured approaches may emerge.

## Approach of This Repository

This repository follows a **problem-first structure**:

```text
1. Observe how AI is used today
2. Identify gaps and limitations
3. Analyze real-world scenarios
4. Introduce structured approaches
```

---

## Key Question

```text
How do we preserve engineering discipline while using AI to accelerate software development?
```
